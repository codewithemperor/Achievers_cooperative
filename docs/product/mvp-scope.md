# MVP Scope

The MVP follows the proposal and prioritizes the fastest path to operational value:

1. Authentication for members and admins
2. Member profile and status management
3. Wallet balance and wallet funding requests
4. Receipt upload and admin approval workflow
5. Basic transaction history
6. Loan application, review, and status tracking
7. Lightweight admin dashboard with pending actions

Deferred to post-MVP:

- richer savings analytics
- advanced investment lifecycle management
- custom report builder
- deeper audit tooling
- production-grade notification orchestration
